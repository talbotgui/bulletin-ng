Conservation du dernier fichier sauvegardé

L'application est en mesure de se souvenir du nom de votre dernière sauvegarde et de vous afficher un avertissment si vous charger un fichier qui n'est pas le dernier modifié.
Mais pour cela, l'application doit être en mesure de stocker ce nom de fichier dans votre navigateur et ce dernier ne doit pas le détruire lors de sa fermeture.

Si cette fonctionnalité n'est pas opérante dans votre navigateur, suivez la procédure ci-dessous :
  • Chrome :
    • Saisir dans la barre d'adresse "chrome://settings/content/cookies"
    • En face de "autoriser", cliquer sur le bouton Ajouter et ajouter le site "https://me.guillaumetalbot.com/maclasse"
    • Pour les développeurs, ajouter aussi "http://localhost:4200"

    Si vous utiliser une extension de Chrome supprimant les données à la fermeture du navigateur, pensez à ne pas supprimer les "local storage data".
